## Swift Student Challenge 자격 요건

​이미 풀타임 개발자로 고용되어 있는 경우 Swift Student Challenge에 지원할 수 없습니다. 또한 앱 플레이그라운드 제출 시점에 다음과 같은 요건을 충족해야 합니다.

​최소 연령 요건: 14세 이상
​Apple Developer로서 Apple에 무료 가입되어 있거나 Apple Developer Program의 멤버인 자
​다음의 요구 사항 중 하나를 충족하는 자
​공인된 교육 기관 또는 이에 준하는 공식 홈스쿨 기관이나 Apple Developer Academy에 등록되었거나 지난 90일 이내에 이를 졸업한 자
​과학·기술·공학·수학 융합 교육(STEM) 기관에 등록된 자
​지난 6개월 이내에 고등학교 또는 그에 준하는 학교를 졸업하고 공인된 상급 교육 기관에 합격했거나 합격을 기다리고 있는 자
​Swift Student Challenge 수상은 최대 4번까지 가능합니다. 우수 수상자로는 1번만 선정될 수 있습니다.

자세한 내용은 [애플 공식 안내](https://developer.apple.com/kr/swift-student-challenge/eligibility/)를 참고해주세요
