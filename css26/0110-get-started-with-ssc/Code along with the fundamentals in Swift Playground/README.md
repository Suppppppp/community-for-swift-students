# Swift Playground 기초 코딩 실습
Code along with the fundamentals in Swift Playground

## 개요

### 난이도

Lv1 | Swift 가 처음인 분 (누구나)

### 세션 정보

1. Xcode의 AI 기능을 활용하여 기본적인 앱 동작을 살펴보고 Swift Student Challenge 아이디어를 구현 단계와 연결하는 방법을 소개합니다

2. 아이디어에 적합한 디바이스(iPad, iPhone, Mac)를 판단하는 기준을 이해하고 초기 프로토타입 제작 과정을 확인해 봅니다

Xcode에서 시작해 Swift Playgrounds로 옮기는 핵심 워크플로우를 이해하고, Swift Student Challenge 준비에 필요한 기본 내용을 습득할 수 있길 바랍니다.

### 필요 환경

아래 중 하나의 앱을 설치해주세요

| 기기 | 앱 |
| --- | --- |
| 맥북 | Swift Playground 4.6+ |
| 맥북 | Xcode 26+ |

### 프롬프트

본 실습은 Xcode에 내장된 AI 기능을 활용하여 간단한 뷰와 레이아웃, 그리고 내비게이션 기능을 구현합니다.

```
1. 빨간색 원, 초록색 사각형, 파란색 삼각형을 화면에 표시하고 기본적으로 세로로 나란히 배치해줘.
2. 화면 하단에 “세로”, “가로” 버튼을 두고, 버튼을 누르면 세 도형의 정렬 방향이 세로와 가로로 전환되도록 해줘.
3. 각 도형을 누르면 해당 도형 하나만 화면 중앙에 표시되는 화면으로 이동하고, 뒤로가기 버튼으로 이전 화면으로 돌아올 수 있도록 해줘.
```
